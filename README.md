# Exam Paper Management System

A web-based Exam Question Paper Management System built for college teachers.

## Tech Stack
- **Python** + **Flask** (backend)
- **OOP** (Question, QuestionBank, ExamPaper classes)
- **Pandas** (search & filter questions)
- **CSV** (question bank storage + paper export)
- **ReportLab** (PDF generation)

## Features
- Add questions to a question bank
- Search/filter by topic and difficulty
- Generate exam papers and save as CSV
- Generate formatted PDF question papers (college header, sections, Bloom's level, CO columns)

## How to Run
```bash
pip install flask pandas reportlab
python app.py
```
Then open http://localhost:5050
